# Tutorials

Welcome to the Blerp tutorials! These guides will help you get started with Blerp and integrate it into your applications.

## Tutorial Series

### [Getting Started](./getting-started/)

Learn how to run Blerp locally for development and testing.

- Setting up your development environment
- Running Blerp with Docker Compose
- Understanding the project structure
- Basic configuration

### [API Usage](./api-usage/)

Learn how to use the Blerp REST API directly.

- Authentication strategies
- Creating and managing users
- Working with organizations
- Session management
- Webhook integration

### [Data Setup](./data-setup/)

Learn how to set up and manage data in Blerp.

- Creating projects and API keys
- Setting up organizations
- Configuring authentication strategies
- Managing user data

### [Next.js Integration](./nextjs-integration/)

Learn how to integrate Blerp into a Next.js application.

- Installing the SDK
- Configuring authentication
- Protecting routes
- Working with sessions
- Server-side user fetching

## Prerequisites

Before starting these tutorials, ensure you have:

- **Bun** v1.2.19+ installed ([install Bun](https://bun.sh))
- Basic knowledge of TypeScript/JavaScript
- Familiarity with REST APIs
- For Next.js tutorials: Node.js 18+ and Next.js 13+

## Getting Help

If you run into issues:

1. Check the [API Reference](/api/) for detailed endpoint documentation
2. Review the [Architecture Guide](/guide/architecture) for system design
3. Open an issue on [GitHub](https://github.com/e6qu/blerp/issues)
